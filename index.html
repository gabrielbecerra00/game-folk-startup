About

GameFolk V1 is a self-contained handheld game console built on a custom PCB. It runs on an MSPM0G3507 microcontroller driving a 128×64 OLED display, has eight tactile buttons, and synthesizes audio through a piezo buzzer. The whole system is powered by three AAA batteries through an onboard regulator, so it runs independently of any debug hardware.

The project demonstrates custom PCB design, SPI communication, GPIO input with internal pull-ups, software square-wave audio synthesis, and use of the RTC peripheral as a wall-clock time source.

Games
  • Pong single-player with position-dependent paddle deflection
  • 2P Pong two players on one device, first to 5 wins
  • Platformer side-scrolling with speedrun timer driven by RTC
  • Tetris all 7 tetrominoes, rotation, NES-style scoring
