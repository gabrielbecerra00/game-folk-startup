GameFolk V1

A Custom Embedded Handheld Game Console

ELEC 327 Final Project · Spring 2026 · Rice University

About

GameFolk V1 is a self-contained handheld game console built on a custom PCB. It runs on an MSPM0G3507 microcontroller driving a 128x64 OLED display, polls eight tactile buttons, and synthesizes audio through a piezo buzzer. The whole system is powered by three AAA batteries through an onboard regulator, so it runs independently of any debug hardware.

Four games are playable from the main menu:

  • Pong — single-player with position-dependent paddle deflection
  • 2P Pong — two players on one device, first to 5 wins
  • Platformer — side-scrolling with a speedrun timer driven by the MSPM0's RTC peripheral
  • Tetris — all 7 tetrominoes, wall-kick rotation, ghost piece, NES-style scoring

The project demonstrates custom PCB design, SPI communication, GPIO input with internal pull-ups, software square-wave audio synthesis, and use of the RTC peripheral as a wall-clock time source.
